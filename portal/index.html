Portal

Portal allows you to create an information portal for use within your Discord Server.

Getting Started

  1. Make a .txt file on your computer with all of the information you'd like. Make sure it is 2048 characters or less
  2. Type @Portal#0301 update and attach and send your *.txt file with it
  3. If you've done everything right, you should now have your portal opened!
Support me on Patreon!

Commands

@Portal help - View Welcome Message

@Portal info - View info on Portal

@Portal getportal - Get a copy of your previously opened Portal

@Portal update - Update your existing Portal or create a new one

@Portal - Views Portal if one exists. Otherwise shows Welcome Message
